Traffic Survey Data Analyzer Project

Posted on October 15, 2025

I recently worked on a Python project that analyzes traffic survey data from CSV files. In this post, I will share what the project does and what I learned.

What Does It Do?

The Traffic Survey Data Analyzer is a console-based tool that processes traffic data from urban junctions. It analyzes vehicle counts, truck percentages, electric vehicles, and rainfall data.

Key Features

Some important features include:

  • Date Validation - Users enter a date and the program validates it
  • Data Analysis - Calculates total vehicles, trucks, and EVs
  • Visualizations - Creates histograms showing hourly traffic patterns
  • Multiple Junctions - Tracks data from different road intersections
  • Rainfall Tracking - Records weather conditions during surveys

How It Works

The program asks for a survey date, finds the matching CSV file, and then analyzes the data. It shows statistics like total vehicles and creates color-coded graphs for each junction.

What I Learned

This project taught me how to work with CSV files, validate user input, and create data visualizations using Python libraries like Pandas and Matplotlib.

Building this tool helped me understand how data analysis can solve real-world problems like traffic planning and congestion management.

My First Year Web Development Group Project

Posted on October 15, 2025

During my first year, I worked on an exciting group project where we built a complete web application. I want to share my experience and what I contributed to the project.

Responsive Sitemap with SVG

I created a responsive sitemap using SVG graphics. This visual representation showed the structure of our website and how different pages connected to each other. The sitemap adjusted beautifully to different screen sizes, making it easy to view on both desktop and mobile devices.

Page Editor

One of my main contributions was building a page editor. This feature allowed users to create and modify content on the website. I learned a lot about handling user input and updating content dynamically using JavaScript.

Validation Page

I developed a validation page that checked user inputs to make sure all information was correct before submitting. This included:


  • Required information checks
  • Error handling

Profile Page

I also built a user profile page where people could view and edit their personal information. The profile page was responsive and included features like profile picture display, editable fields, and a clean user interface.

What I Learned

This group project taught me valuable skills in teamwork, responsive design, and building functional web applications. Working with my team helped me understand how different parts of a website work together.

I am proud of what we accomplished and excited to build more complex projects in the future!
